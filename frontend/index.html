MiniLang Syntax Guide

      

        
Variables

        let x = 10;

        let y;  // defaults to 0

        x = x + 1;

        
Print

        print x;

        
Control Flow

        if (x > 5) { ... } else { ... }

        while (x < 10) { ... }

        for (let i = 0; i < 5; i = i + 1) { ... }

        switch (x) { case 1: ... case 2: ... default: ... }

        break; continue;

        
Expressions

        + - * / % == != < <= > >= && || !

        condition ? value1 : value2

        true false numbers (3.14) strings ("hello")

        (expr)

        read()  // reads a value from user input
